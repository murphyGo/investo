## Dated Primary-Source Evidence

Checked on 2026-07-19:

1. `https://docs.alpaca.markets/us/docs/about-market-data-api`
   - documents free Basic, US stock/ETF coverage, history since 2016, 200 historical
     calls/minute, the latest-15-minute limitation, and key/secret authentication.
2. `https://docs.alpaca.markets/us/reference/stockbars`
   - documents the multi-symbol historical-bars endpoint, `1Day` cadence,
     pagination, feed selection, and raw/split/dividend/spin-off/all adjustments.
3. `https://alpaca.markets/support/redistribute-alpaca-api`
   - answers that Alpaca API data cannot be redistributed.
4. `https://files.alpaca.markets/disclosures/library/AcctAppMarginAndCustAgmt.pdf`
   - section 30 requires written Alpaca consent before market data is reproduced,
     distributed, sold, or commercially exploited.

The source is technically stronger than Finnhub for u140 because the required history
is available on a free plan. It nevertheless fails the binding public-rights gate.
Investo does not infer that a derived sector score is permitted when the provider's
official support rejects API-data redistribution and no written derived-display grant
is present.

## Deduplication Evidence

A repository scan across `src/`, `tests/`, `scripts/`, `.github/`, `CONTRIBUTING.md`,
dependency configuration, AIDLC plans, and requirements found no Alpaca adapter,
registration, source spec, tier/window/segment route, fixture, credential, or
workflow. Adding it would introduce a new two-secret trust boundary and would still
leave the public rights gate unresolved.

## Gate Decision

| Binding gate | Result | Reason |
| --- | --- | --- |
| Official terms and public derived-display rights | Fail | Redistribution prohibited; written consent absent; no explicit derived-display grant |
| Free access for 63+ daily OHLCV bars | Preliminary pass | Basic is free, covers US stocks/ETFs since 2016, and exposes daily multi-symbol bars |
| Exact 12-symbol coverage and freshness | Not run | Fail-fast after the binding rights failure |
| Five bounded GitHub Actions probes | Not run | A new credential path and provider payload testing are unjustified after rejection |
| No paid plan/trial/secret exposure | Partial | No paid plan required, but a new key/secret pair would be required |
| Public/raw retention contract | Not established | No public-use grant; no payload retained |

**Disposition: `reject under current written terms`.** Reconsider only with written
permission that explicitly covers Investo's public derived fields, attribution,
caching, and raw-retention policy. No account/key, local probe, GitHub Actions probe,
adapter, fixture, Pages output, or TECH-DEBT item is created.
