## Dated Primary-Source Evidence

Checked on 2026-07-19:

1. `https://finnhub.io/docs/api/stock-candles`
   - labels stock candles as Premium and documents daily split adjustment;
   - documents token authentication and the OHLCV response fields.
2. `https://finnhub.io/pricing-stock-api-market-data`
   - lists paid market-data plans, daily OHLC history, request limits, and
     `Personal Use` licenses.
3. `https://finnhub.io/pricing`
   - lists a USD 0 general plan but leaves US-market OHLC unavailable.
4. `https://finnhub.io/register`
   - requires the registrant to confirm qualified non-professional/personal use or
     obtain written approval as a commercial/professional user.

These official surfaces are sufficient to reject the candidate. Public display,
derived redistribution, attribution, caching, and raw-retention terms do not need to
be inferred or tested after the provider already fails the no-paid gate and lacks the
required written public-use approval.

## Deduplication Evidence

A repository scan across `src/`, `tests/`, `scripts/`, `.github/`, `CONTRIBUTING.md`,
and dependency configuration found no Finnhub adapter, registration, source spec,
tier/window/segment route, fixture, secret, or workflow. The only prior references
were planning candidates in `docs/requirements.md` and the u138 story map.

Adding Finnhub would not repair the u140 gap: it would introduce a new paid,
credentialed source while the existing Yahoo path remains separately deferred for
public-rights reasons.

## Gate Decision

| Binding gate | Result | Reason |
| --- | --- | --- |
| Official terms and public derived-display rights | Fail | Personal Use license; no Investo-specific written public/commercial approval |
| Free access for 63+ daily OHLCV bars | Fail | Stock-candle endpoint is Premium-only |
| Exact 12-symbol coverage and freshness | Not run | Fail-fast after binding rights/cost failures |
| Five bounded GitHub Actions probes | Not run | Credentialed paid-source probing is out of scope after rejection |
| No paid plan/trial/secret exposure | Fail | Paid market-data subscription and API token required |
| Public/raw retention contract | Not established | No public-use grant; no payload retained |

**Disposition: `reject`.** No source implementation, credential request, local raw
probe, GitHub Actions workflow, fixture, Pages output, or TECH-DEBT item is created.
u140 remains blocked and the next candidate must begin with a new Step 0 fact sheet.
