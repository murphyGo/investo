# u140 Source Fact Sheet — EODHD EOD API

**Checked**: 2026-07-19
**Candidate**: EODHD End-Of-Day Historical Stock Market Data API
**Disposition**: Reject under current written terms for public Pages
**Probe status**: Not run; the candidate failed the public-rights gate first

## Dated Primary-Source Evidence

Checked on 2026-07-19:

1. `https://eodhd.com/financial-apis/api-for-historical-data-and-volumes`
   - documents the single-symbol token-authenticated `/api/eod/{ticker}` endpoint,
     daily period, CSV/JSON formats, date ordering, and OHLC/adjusted-close/volume
     fields;
   - gives free accounts 20 calls/day and one year of EOD history for any ticker;
   - identifies `close` as unadjusted and `adjusted_close` as split- and
     dividend-adjusted, and states that major US exchanges update within 15 minutes
     after close.
2. `https://eodhd.com/pricing`
   - lists Free Starter at USD 0, 20 calls/day, 20 requests/minute, one year of data,
     and Personal use;
   - directs commercial users to a separate commercial solution rather than
     granting public display under the free plan.
3. `https://eodhd.com/financial-apis/terms-conditions`
   - permits Non-Professional Users to store, manipulate, and analyze Information
     only for private, non-commercial purposes;
   - prohibits selling, reselling, retransmitting, redistributing, displaying, or
     granting access to Information or Services in original or repackaged form;
   - requires Professional Users to obtain prior written approval for display or
     redistribution and requires stored data deletion within one month after the
     subscription ends.
4. `https://eodhd.com/financial-apis/our-data-sources-and-data-partners`
   - identifies Nasdaq Cloud API as the source for US-market data and describes the
     broader exchange/provider provenance.

EODHD is technically sufficient for a once-daily 12-symbol collector, but the free
license is explicitly private and personal. Investo Pages is accessible to other
users and publishes computed sector analytics, so neither repackaging the data nor
retaining only derived values creates permission under the current terms.

## Deduplication Evidence

A repository scan across `src/`, `tests/`, `scripts/`, `.github/`, `CONTRIBUTING.md`,
dependency configuration, AIDLC plans, and requirements found no EODHD,
EODHistoricalData, or `/api/eod` adapter, registration, source spec,
tier/window/segment route, fixture, credential, dependency, or workflow. Adding it
would introduce a new query-token trust boundary while leaving the public-use gate
failed.

## Gate Decision

| Binding gate | Result | Reason |
| --- | --- | --- |
| Official terms and public derived-display rights | Fail | Free access is private/non-commercial and display or redistribution is prohibited; professional display requires prior written approval |
| Free access for 63+ daily OHLCV bars | Preliminary pass | One free year exceeds 63 bars, the endpoint exposes OHLCV plus adjusted close, and 12 requests fit the 20-call daily quota |
| Exact 12-symbol coverage and freshness | Not run | Fail-fast after the binding rights failure; the documented US update window alone is not probe evidence |
| Five bounded GitHub Actions probes | Not run | A new account, secret, and provider-payload testing are unjustified after rejection |
| No paid plan/trial/secret exposure | Partial | Free access is technically sufficient, but it requires a secret token and grants only Personal use |
| Public/raw retention contract | Fail | Public display/redistribution is prohibited and post-subscription deletion is mandatory |

**Disposition: `reject under current written terms`.** Reconsider only with EODHD's
prior written approval covering Investo's public derived fields, attribution,
caching, raw-retention policy, and GitHub Pages delivery on a no-paid basis. No
account/key, local probe, GitHub Actions probe, adapter, fixture, Pages output, or
TECH-DEBT item is created.
