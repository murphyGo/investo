## Dated Primary-Source Evidence

Checked on 2026-07-19:

1. `https://www.tiingo.com/documentation/end-of-day`
   - documents token-authenticated metadata and historical-price endpoints;
   - defines raw/adjusted OHLCV, dividend, and split fields, JSON/CSV responses,
     CRSP-style split/dividend adjustments, and the 5:30-8:00 PM EST update window.
2. `https://www.tiingo.com/pricing`
   - lists free Starter at 500 unique symbols/month, 50 requests/hour, 1,000/day,
     and 1 GB/month;
   - classifies the listed standard plans as Internal Use Only and explains that
     internal use excludes displaying or sharing data with another person or
     organization.
3. `https://www.tiingo.com/products/end-of-day-stock-price-data`
   - documents broad equity/ETF coverage, raw and adjusted OHLCV, history, and EOD
     cadence;
   - lists display redistribution from USD 250/month for startups and USD 500/month
     for enterprise use.
4. `https://app.tiingo.com/tos/`
   - last updated 2026-02-18, states API data is for internal consumption only,
     prohibits publishing or publicly making available analysis of Tiingo data,
     and requires special permission plus additional fees for redistribution.

The free plan is technically sufficient for one request per symbol per day. It does
not clear the product boundary: Investo's public radar is a public analysis/display,
not internal consumption, and the only explicit redistribution path is paid. The
attribution requirement applies after permission and does not substitute for it.

## Deduplication Evidence

A repository scan across `src/`, `tests/`, `scripts/`, `.github/`, `CONTRIBUTING.md`,
dependency configuration, AIDLC plans, and requirements found no Tiingo adapter,
registration, source spec, tier/window/segment route, fixture, credential,
dependency, or workflow. Adding it would introduce a new token trust boundary and
would still violate u140's no-paid public gate.

## Gate Decision

| Binding gate | Result | Reason |
| --- | --- | --- |
| Official terms and public derived-display rights | Fail | Internal consumption only; public analysis/display prohibited without special permission |
| Free access for 63+ daily OHLCV bars | Preliminary pass | Free Starter has broad EOD coverage and ample per-day/per-month limits |
| Exact 12-symbol coverage and freshness | Not run | Fail-fast after the binding rights and no-paid failures |
| Five bounded GitHub Actions probes | Not run | A new token and provider-payload testing are unjustified after rejection |
| No paid plan/trial/secret exposure | Fail | Public display redistribution is explicitly priced from USD 250/month |
| Public/raw retention contract | Not established | No free redistribution permission; no payload retained |

**Disposition: `reject`.** Reconsider only if Tiingo supplies a free written license
that explicitly covers Investo's public derived fields, attribution, caching,
raw-retention policy, and GitHub Pages delivery. No account/token, local probe,
GitHub Actions probe, adapter, fixture, Pages output, or TECH-DEBT item is created.
