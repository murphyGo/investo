# Vision Document: Investo

## Project Overview

Investo는 매일 자동으로 미국 주식·크립토(보조: 코스피) 시장의 데일리 시황을 한국어로 생성·게시하는 개인용 자동화 도구다. 무료 공개 데이터 소스에서 전일 핵심 이슈·섹터 동향·주요 지표·실적·종목 뉴스를 수집하고, Claude Code CLI를 통해 일관된 7섹션 포맷의 시황을 작성한 뒤, GitHub Pages에 영구 게시하고 텔레그램으로 요약을 푸시한다.

## Goals

1. 매일 직접 뉴스를 검색·정리하는 시간 비용을 0으로 만든다
2. 일관된 형식의 시황을 통해 시점별 시장 흐름을 추적 가능하게 한다
3. 운영비 월 $0 (Claude Code 구독 외) 유지
4. 1인 운영자가 단순한 plugin 추가만으로 데이터 소스를 확장할 수 있는 구조 유지
5. 향후 포트폴리오 트래킹·기업 분석 기능을 단계적으로 추가할 수 있는 토대 제공

## Scope

### In Scope (MVP)
- 무료 데이터 소스 다수 통합 수집 (뉴스, 시세, 거시 지표, 연준 캘린더, 실적 캘린더)
- Claude Code CLI 기반 한국어 시황 자동 생성 (7섹션 정형 포맷 + 면책조항)
- GitHub Pages 정적 사이트 게시 (날짜별 archive)
- 텔레그램 Bot API 알림 (요약 + 링크)
- GitHub Actions cron 스케줄 실행 (평일 KST 07:00 + 토요일)
- 실행 실패 알림 (텔레그램 + GitHub Actions notification)

### Out of Scope (MVP)
- 실시간 시황·알람
- 사용자 계정·인증·구독 관리
- 포트폴리오 트래킹 (향후 확장)
- 기업 펀더멘털 심층 분석 (향후 확장)
- 모바일 네이티브 앱
- 자동 매매
- 유료 데이터 소스
- 한국어 외 언어 시황
- 다중 사용자 운영

## Stakeholders

| Role | Identity | Interest |
|------|----------|----------|
| Primary user | 본인 (개인 투자자) | 매일 시황으로 시장 파악 |
| Secondary readers | 본인이 공유하는 지인 | public GitHub Pages 열람 |
| Operator | 본인 1인 | 코드 유지·데이터 소스 추가 |

## Success Criteria

- 매일 정해진 시각에 시황 자동 생성·게시 (실패율 ≤ 5%)
- 시황의 핵심 이슈 누락·사실 오류 ≤ 일 1건
- 운영자가 매일 시황을 실제로 읽고 참고 (정성적)
- 월 운영비 $0
- 데이터 소스 신규 추가 1건당 코드 변경 ≤ 1 모듈
